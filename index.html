Generación de números aleatorios

Este método Utiliza la función trigonométrica del Coseno aprovechando que es una función tracendente para la gneracion de números aleatorios, tomando semilla de 4 digitos, ésta se toma como ángulo para evaluarla en el Coseno, al resultado se convierte en valor absoluto al estar el domminio de la función entre -1 y 1, a este resultado se extraen 4 digitos iniciando desde el 4to decimal hasta el 8vo para la nueva semilla esta se divide entre 10000 para así obtener el numero aleatorio.
